# Workout-Planner-System
Project Description

The Workout Planner System is a Java Swing GUI application that helps users manage their daily workouts. Users can add workout details, view all workouts, search for specific workouts, calculate total workout duration, and clear workout records.

This project is designed to demonstrate the use of Core Java concepts such as OOP, Collections, GUI development, and Event Handling.

🚀 How It Works
Add Workout

The user enters:

Workout Name
Duration (minutes)
Calories Burned

The workout is stored in an ArrayList.

View Workouts

Displays all workouts added by the user.

Search Workout

Searches for a workout using its name.

Total Duration

Calculates the sum of durations of all workouts.

Clear All

Removes all workout records from the system.
